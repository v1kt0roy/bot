## 开发缘由
>本bot最开始应用于nanahira的粉丝群,用于推送曲子与群友互动,原本基于酷q+http-api,在酷q停止运营后转移使用[mirai-http-api](https://github.com/project-mirai/mirai-api-http)和[graia](https://github.com/GraiaProject),目前仍在完善功能。
